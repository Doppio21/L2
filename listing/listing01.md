Что выведет программа? Объяснить вывод программы.

package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}

Данная программа выведет [77 78 79], так как созается слайс b, 
копирующий элементы массива а с индексом от 1 до 3 включительно.
